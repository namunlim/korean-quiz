📊 Google 表單整合

設定後，學生答題完成提交時，成績會自動送到你的 Google 表單試算表，你可以隨時查看所有學生的成績。

📋 設定步驟

請先依照以下步驟在 Google 表單取得必要資訊，再填入下方欄位。

  1. 前往 forms.google.com，建立新表單，新增 4 個「簡答」欄位：
    學生姓名、練習題名稱、總分、答題詳情
  2. 點右上角 ⋮ → 取得預先填入的連結，每格填任意字，點「取得連結」
  3. 複製網址，找出 entry.xxxxxxxxx 格式的 4 個欄位 ID
  4. 將 viewform 改成 formResponse 記下表單 ID（網址中間那段長字串）
  5. 表單設定 → 取消「限制為組織內部」，確保任何人都能提交

🚀 GitHub Pages 部署指引

把匯出的 HTML 上傳到 GitHub Pages，學生就能用手機或電腦直接開啟網址答題，不需要下載任何東西。

📖 完整操作步驟

全程免費，約 5 分鐘完成設定。之後每次更新題目只需重新上傳 HTML 即可。

1
建立 GitHub 帳號
前往 github.com 點「Sign up」免費註冊，記住你的帳號名稱
2
建立 Repository
登入後點右上角 + → New repository，名稱輸入 korean-quiz，設為 Public，勾選「Add a README file」，點「Create repository」
3
上傳練習題 HTML
進入 repository 後點「Add file」→「Upload files」，把匯出的 HTML 拖進去，將檔名改成 index.html，點「Commit changes」
4
開啟 GitHub Pages
進入「Settings」→ 左側選「Pages」→ Source 選「Deploy from a branch」→ Branch 選 main，選 /(root)，點「Save」
5
等約 1 分鐘後你的網址就是：
https://【你的GitHub帳號名稱】.github.io/korean-quiz/
把這個連結傳給學生，手機電腦都能直接打開答題 🎉
↻
之後更新題目
重新匯出 HTML → 上傳到 GitHub 同一個位置覆蓋 index.html → 約 1 分鐘後網址自動更新
